A course browser for Illinois Tech

Pop replaces Banner's complex interface with one simple search box, except there's no search button because searching is instant. It searches all courses available for the Fall 2016 semester, just type something in the orange bar to get started! You can type pretty much anything, including:

  • CRNs: 10564
  • Places: siegel
  • Professors: twombly
  • Departments: itm
  • Course titles: comp graphics
  • Days of the week: mwf
  • Course descriptions: leadership
  • Course abbreviations: econ211
  • Any combination of the above: hist comp mw
